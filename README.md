# 🎬 CineConnect

> Find verified movie companions in your city. Browse outings, join a group, book tickets together.

## Live Site
🔗 Coming soon — deploy via Vercel (see below)

## Tech Stack
- **React 18** + **Vite 5**
- No backend required for prototype (in-memory state)
- Ready to connect: Supabase (DB) + Clerk (Auth) + Twilio (SMS)

## Run Locally
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Deploy to Vercel (Free)
1. Push this repo to GitHub
2. Go to https://vercel.com → Import repo
3. Framework: Vite (auto-detected)
4. Click Deploy ✅

## Features
- 🔍 Browse movie outings by genre, city, verified status
- ✏️ Post your own outing with safety agreement
- 🎟️ Book tickets at Cineplex, AMC, IMAX, Landmark, Fandango
- 📊 Dashboard with genre/cinema/city analytics
- 🛡️ Safety: verification badges, community reporting, admin queue
- 🚀 Interactive launch checklist built in

## Safety
All hosts verified · Public lobby meetups only · 18+ platform · Community reporting · Zero tolerance policy
